# Introduction

基础知识，建立基本概念，补全从入门到精通的知识空缺。

### [vector_add](./vector_add)
演示如何使用 AscendC 框架在 NPU 上编写简单的 Vector Core kernel，实现向量逐元素相加。入门级向量计算示例。

### [vector_add_c_api](./vector_add_c_api)
演示如何使用 Ascend C API（C语言风格接口）在 NPU 上编写简单的 Vector Core kernel，实现向量逐元素相加。入门级向量计算示例。支持架构：dav-2201。

### [matmul](./matmul)
演示如何使用 AscendC 框架在 NPU 上编写简单的 Cube Core kernel，实现矩阵乘法运算。入门级矩阵计算示例。
